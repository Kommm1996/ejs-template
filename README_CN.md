# ejs-template

 [English](./README.md)

 使用ejs编译html的模板

## 快速开始

推荐使用pnpm，如果不想使用pnpm，请先删除package.json中的以下代码。

``` json
"preinstall": "npx only-allow pnpm"
```

### 安装依赖

``` bash
# pnpm
pnpm i
# npm
npm i
```

### 开发模式

``` bash
# pnpm
pnpm dev
# npm
npm run dev
```

打开 ```localhost:8080```

### 生产模式

``` bash
# pnpm
pnpm build
# npm
npm run build
```

生产模式执行后的文件在/dist目录下
